removeItemButton
directive in module ui.listInput

Description

Replaces the element on which the directive is applied with a standard .input-group-addon button for removing the item under which it appears in the list. Used internally in uiListInput but may also be used in custom fields for uniformity and convenience.

Usage

as element:
<remove-item-button>
</remove-item-button>
as attribute
<ANY remove-item-button>
   ...
</ANY>
as class
<ANY class="remove-item-button">
   ...
</ANY>
